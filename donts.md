# AltShift Web — プロジェクト固有 Don'ts

`global-donts.md` に加えて、altshift.jp で追加で禁止する事項。

---

## ブランド / コンテンツ

- 「AltShift」のロゴ・表記を変更しない（大文字小文字含む）
- 「ディーチャー」「DTEACHER」の表記を統一から外さない
- サービス料金を確認なしで変更しない（営業に直結する）
- クライアント企業名を実績ページに掲載する場合はDaiki氏の許可が必要
- 「CAIO」の定義・説明文を勝手に書き換えない（ポジショニングの根幹）

## 技術

- Astro + Tailwind の構成を勝手にReactに移行しない
- ドメイン（altshift.jp）のDNS設定を変更しない
- Vercelの本番環境変数を確認なしで変更しない
- SEO関連のmeta情報（title, description, OGP）を確認なしで変更しない

## 営業 / 外部連携

- 問い合わせフォームの送信先を変更しない
- 料金表の価格を値下げ方向に変更しない（値上げもDaiki氏確認）
- 競合他社の社名を明示的に記載しない
